Open Tree of Life Source Data

Synthesis Source Trees

This site provides downloads of the processed source tree files used as input data for v4.0 of the Open Tree of Life synthetic tree. Trees are in Newick format.

There are 484 distinct source trees (originating from 478 studies). Tree files are named as:

prefix_studyid_treeid

These trees differ in three ways from their versions available from the Open Tree Study Curator:

  1. Trees include only the ingroup clade designated by study curators.
  2. Within a tree, duplicate taxa are pruned to one exemplar each.
  3. Tip labels correspond to taxon ids from version 2.9draft12 of the Open Tree Taxonomy (OTT).
Unprocessed versions of each tree can be obtained from:

https://tree.opentreeoflife.org/curator/study/view/<PREFIX>_<STUDYID>
